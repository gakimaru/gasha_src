ライブラリソースリポジトリ
---
このリポジトリは、「GASHA」のソースファイル用です。  
> 共同開発メンバーに対して、ソースコードを非公開でライブラリを提供するような場面を想定し、ライブラリ本体（インクルードファイルとライブラリファイル）とリポジトリを切り分けて扱っています。  
> 
> ライブラリファイル作成用のプロジェクト／Makefileも扱っています。
